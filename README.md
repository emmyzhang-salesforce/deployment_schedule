## deployment_schedule

Example Express App for displaying deployment schedules

#### Tools Used:

* ExpressJS for routing
* EJS for templating

#### To Do:

## Installation

1. Clone repo: ```git clone git@github.com:emmyzhang-salesforce/deployment_schedule.git ```
2. Run ```npm install```
3. Run ```npm start```
4. Go to ```localhost:3030```
